# Security Policy

## Supported Versions

The following versions of Pill are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via GitHub Issues or contact the maintainers directly.

We take security seriously and will investigate all reports. Please do not disclose vulnerabilities publicly until a fix has been released or we've had a chance to respond.
